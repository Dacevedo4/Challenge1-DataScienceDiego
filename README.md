# Challenge1-DataScienceDiego
Este repositorio contiene el análisis completo de ventas de cuatro tiendas, realizado para determinar cuál es la mejor opción para enfocar las ventas del Sr. Juan. Incluye el informe final, con un estudio de ingresos, popularidad de categorías, satisfacción de clientes, productos más y menos vendidos y costos de envío promedio. El objetivo es proporcionar información clara y objetiva para la toma de decisiones estratégicas.
